⚙️ Configurar base de datos

Conecta Firebase para que todos los usuarios vean las tareas en tiempo real desde cualquier dispositivo. Solo se hace una vez.

1Ve a console.firebase.google.com e inicia sesión con Gmail.
2Clic en "Agregar proyecto" → ponle un nombre → continuar.
3Ve a Realtime Database → Crear base de datos → Modo de prueba → Habilitar.
4Ve a Configuración ⚙️ → Tus apps → Web. Registra la app y copia los datos de configuración.
5Pega los valores aquí abajo y haz clic en Guardar y conectar.
